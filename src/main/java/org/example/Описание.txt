Проанализировать и описать персонажей: Маг, монах, разбойник, копейщик, снайпер, арбалетчик, крестьянин.
На базе описания персонажей описать простейшую иерархию классов. В основной программе создать по одному экземпляру каждого класса.

Базовые свойства:
Здоровье
Броня
Инициатива
Урон
Дальность поражения

Маги:
Мана

Лучники:
Кол-во боеприпасов

Крестьянин:
Грузоподъемность

Базовые действия:
Передвижение
Атака (по умолчанию каждый может атаковать в ближнем бою)
Защита

Монах:
Медитация(+мана)
Лечение(-мана)

Маг:
Файербол(-мана)

Снайпер:
Выстрел
Дальний выстрел(доп. дальность)

Арбалетчик:
Выстрел
Перезарядка (пропуск хода)
Прицельный выстрел(доп. урон)

Разбойник:
Рывок(доп. передвижение)
Крит. удар(доп.урон)

Копейщик:
Таран(передвижение+урон)
